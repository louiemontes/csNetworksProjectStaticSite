About


Team Intro

Our team is composed of Austin Kelly, Brandon Kindrick and Luis Montes. We are all currently undergrads at NAU studying Computer Science. The project was created for an assignment in "CS499: Network Analysis/Characterization" with Dr. Vigil-Hayes. We decided to research the topic we all shared an interest in board games. We came to this decision, because we all shared the common interest in board games and because we realized there was a lot of potential for research topics.


High Level Description

Our project aims to aid all individuals who are interested in discovering new board games relevant to their interests. This can be helpful to those who are very experienced with board games, or for individuals new to the hobby but don't know where to start. We’ve aimed to accomplish this by analyzing the top 50 board games (provided by Board Game Geek), with Jaccard Similarity Index. We chose this metric, because it takes in two sets of data and returns a threshold of how similar the two sets are. This is useful to our research because it can be used to compare board games based on two metrics, category and mechanics. The Jaccard Similarity Index returns the most similar board games from the list of top 50 board games. We also aim to implement a web app that would take in user input referencing a board game, and then output the most similar games from the list of top 50 most popular board game.
