About

Our team is composed of Austin Kelly, Brandon Kindrick and Luis Montes. We are all currently undergrads at NAU studying Computer Science. The project was created for an assignment for "CS499: Network Analysis/Characterization" with Dr. Vigil-Hayes. We decided to research the topic we all shared an interest in board games. We came to this decision, because we all shared the common interest in board games and because we realized there was a lot of potential for research topics.
